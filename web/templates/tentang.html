Tentang Aplikasi SIMDOKPOL

Sistem Informasi Manajemen Dokumen Kepolisian

Versi: 1.0.0

SIMDOKPOL adalah sebuah aplikasi web yang dirancang untuk membantu unit kepolisian dalam manajemen dan penerbitan surat keterangan secara efisien, cepat, dan aman. Aplikasi ini dibangun dengan arsitektur modern dan dapat berjalan sepenuhnya secara offline tanpa memerlukan koneksi internet.

Fitur Unggulan:
  • Manajemen Dokumen dan Pengguna yang komprehensif.
  • Dasbor analitik untuk pemantauan data secara real-time.
  • Formulir cerdas dengan sistem rekomendasi petugas.
  • Konfigurasi sistem yang dinamis melalui alur setup terpandu.
  • Modul Audit Log untuk melacak semua aktivitas penting.
  • Fitur Backup & Restore database dengan path yang bisa diatur.
  • Pratinjau cetak dengan presisi tinggi sesuai format fisik.
Tumpukan Teknologi:
  • Backend: Go (Golang) dengan framework Gin
  • Database: SQLite
  • Frontend: HTML, CSS, JavaScript dengan jQuery & Bootstrap
